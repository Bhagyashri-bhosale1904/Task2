# Task2
This repository contains the source code for my Task 2.
Temperature Converter 
Overview
This program converts temperatures from Fahrenheit to Celsius. It demonstrates both static conversion using a predefined value and dynamic conversion using user input.

Features
-Static conversion with a fixed Fahrenheit value
-Dynamic conversion by reading user input with the Scanner class
-Demonstrates arithmetic operations and console output

Concepts Covered
-Variable declaration (double fahrenheit)
-Arithmetic operators (-, *, /)
-User input handling with Scanner
-Console output using System.out.println and System.out.printf
